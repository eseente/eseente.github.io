TicTacToe Game

It's an unusual tictactoe game. We can play offline or online. Game has three different features which are "magic", "bomb" and players visibility. What I mean by players visibility is, players cannot see each other's moves. However, thanks to magic feature players can make visible four square for three seconds. Players can clear any square with using bomb feature. We can play online and keep player's latest games, leader board and other player informations using Firebase Real-Time Database. Also, we have authentication with phone number. This project made with React Native CLI and Firebase Real-Time Database.

It also saves user data and game history, displaying them on a leaderboard. Unlike the traditional Tic-Tac-Toe game, this version features a 3x3 playing area on each face of a three-dimensional cube. Players must continue the game by considering and keeping track of each face of the cube.

Book Store

This is a project I developed with React.js for the frontend and NestJS with PostgreSQL for the backend. The project includes a different interface for admin users. Regular users can add books to their cart, place orders, and search for books by title and ISBN. After completing the project, I deployed it to a server on DigitalOcean.
